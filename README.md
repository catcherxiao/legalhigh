# 法槌槌 - AI法律助手

一个基于人工智能的法律服务平台，提供智能法律问答、合同风险分析和智能合同生成等功能。

## 功能特性

### 1. AI法律问答
- 实时智能对话
- 支持法条引用和解释
- Markdown格式化输出
- 支持代码块和表格展示
- 聊天记录持久化存储

### 2. 合同风险分析
- 支持多种文件格式上传 (PDF、Word、图片)
- 智能识别合同文本
- 多维度风险评估:
  - 风险等级评定
  - 主要风险点识别
  - 改进建议生成
  - 合规性检查
- 分析报告导出功能

### 3. 智能合同生成
- 支持多种合同类型:
  - 房屋租赁合同
  - 物品租赁合同
  - 劳动合同
  - 服务合同
  - 买卖合同
  - 生产合同
- 分步表单填写
- 实时合同预览
- 支持在线编辑
- HTML格式导出
